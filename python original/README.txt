App takes screenshot of your current screen used. Select region, where you want to detect a text.
Then, change image processing value to better recognize the text. 

Make sure, that path.txt contains correct path to the tesseract.exe file with double slash ("\\")

<Controls>:
Press <ESC> or press X to exit.
Press <SPACE> to move to the next step.
